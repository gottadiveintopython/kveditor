### 実行に必要なもの(requirements)

- Python
- Kivy
- Pygments

### Test環境(Test Environment)

- Python 3.4.3 + Kivy 1.9.1  
- Python 2.7.6 + Kivy 1.9.1  
- Kivy Launcher (Android App)

### Keyboard Shortcuts (doesn't work on android)

- Ctrl + U = Update
- Ctrl + S = Save
- Ctrl + L = Load
