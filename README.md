A simple Kv language editor

### 実行に必要なもの(requirements)

- Python3
- Kivy
- Pygments

### 開發環境(Test Environment)

Python3.4.3  
Kivy1.9.1

### Keyboard Shortcuts

以下のShortcutkeyが使えます。

- Ctrl + U = Update
- Ctrl + S = Save
- Ctrl + L = Load
